[![travis][4]][5]

## Installation

```shell
npm install --save-dev eslint \
  eslint-plugin-patternplate
```

## Usage

Configure eslint to use `babel-eslint` and `eslint-plugin-patternplate`.
Make sure to enable the plugin via its rule.

```json
{
  "eslintConfig": {
    "plugins": ["patternplate"]
  }
}
```
